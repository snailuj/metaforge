# Metaforge Feature Designs

Feature-level brainstorms and design documents. These accumulate during design phase, then synthesize into `Metaforge-Build-Map.md` once all major features are designed.

## Completed Designs

- [metaphor-forge.md](./metaphor-forge.md) - Sprint Zero: property extraction, matching algorithm, tiers, Grimoire
- [string-handling.md](./string-handling.md) - Cross-cutting: Fluent-based UI strings, versioned caching, locale stubs
- [core-thesaurus.md](./core-thesaurus.md) - Search, results panel, copy-to-clipboard, strategy pattern rendering

## In Progress

- [curated-property-vocabulary.md](./curated-property-vocabulary.md) - Two-pass snap-to-vocabulary for Forge matching, antonym contrast tiers

## Pending Designs

- `3d-visualization.md` - Semantic solar system, antique theme, planet rendering
- `word-hunt.md` - Daily quest, clue system, scoring, streaks
- `constellation.md` - Personal journey map, minimap, navigation

## Design Process

1. Brainstorm feature in conversation
2. Save to `docs/designs/{feature-name}.md`
3. Once all major features designed, update `Metaforge-Build-Map.md`
4. Then move to implementation with TDD
